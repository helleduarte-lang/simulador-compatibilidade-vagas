Simulador de Compatibilidade de Vagas

- Sobre o projeto:

O Simulador de Compatibilidade de Vagas é um projeto desenvolvido em Python com Streamlit que simula o funcionamento básico de sistemas de triagem automatizada de currículos (ATS).
A ferramenta permite comparar o texto do currículo com a descrição de uma vaga e calcular um percentual aproximado de compatibilidade, destacando habilidades encontradas e habilidades faltantes.

O projeto foi desenvolvido como parte de um processo de aprendizado em Python iniciado durante uma imersão da Alura e aprimorado com estudos práticos e apoio do ChatGPT.

- Funcionalidades:

Comparação automática entre currículo e descrição de vaga

Identificação de habilidades encontradas e faltantes

Cálculo de compatibilidade ponderada por tipo de skill

Sugestões de melhoria do currículo com base nas habilidades faltantes

Interface interativa construída com Streamlit

- Como executar o projeto:

Clone o repositório:

git clone <link-do-repositorio>

Acesse a pasta do projeto:

cd nome-do-projeto

Ative o ambiente virtual:

.venv\Scripts\activate

Instale as dependências:

pip install -r requirements.txt

Execute o aplicativo:

python -m streamlit run app.py

Abra o navegador no endereço exibido no terminal (normalmente http://localhost:8501)
